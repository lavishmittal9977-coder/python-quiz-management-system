# 🐍 Python Quiz Management System

A console-based Quiz Management System built using Python. The application allows users to log in, attempt a quiz with randomized questions, calculate scores, and automatically receive their quiz results via email using Gmail SMTP.

---

## ✨ Features

- 🔐 User Login System
- 🎲 Randomized Quiz Questions
- 📊 Automatic Score Calculation
- 🏆 Pass / Fail Result
- 📧 Automatic Result Email using Gmail SMTP
- 🧩 Modular Programming
- ⚠️ Exception Handling
- 📂 Clean Project Structure

---

## 🛠️ Technologies Used

- Python
- SMTP (Email Automation)
- Random Module
- Functions
- Custom Modules
- Lists, Dictionaries & Sets

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Add your Gmail email and App Password in `email_sender.py`.
4. Run:

```bash
python main.py
```

---

## 📧 Output

- User logs in
- Quiz starts
- Score is calculated
- PASS/FAIL is displayed
- Result email is sent automatically
