# 🐍 Python Quiz Management System

> A console-based Quiz Management System built with **Python** featuring user authentication, randomized quiz questions, score calculation, and automatic email notifications using Gmail SMTP.

---

## 🚀 Features

- 🔐 User Login Authentication
- 🎲 Randomized Quiz Questions
- 📝 Real-time Score Calculation
- 🏆 PASS / FAIL Result Generation
- 📧 Automatic Result Email using Gmail SMTP
- 🧩 Modular Programming using Custom Python Modules
- ⚠️ Exception Handling
- 📂 Clean and Organized Project Structure

---

## ⚙️ How to Run

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-quiz-management-system.git
```

### Open the project

```bash
cd python-quiz-management-system
```

### Configure Email

Open **email_sender.py**

Replace

```python
sender_email = "your_email@gmail.com"
app_password = "your_app_password"
```

with your Gmail credentials (use a Gmail App Password).

### Run

```bash
python main.py
```
